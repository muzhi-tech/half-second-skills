# The Half Second — Script Builder (ARISE)

A working reference for turning a desired behavior change into an
**installable script**, based on Li Xiaolai's *The Half Second* (2026).
Mechanism: System 1 has a **frequency counter** that *counts, does not
check* — repetition of a chosen sentence, in the speaker's own voice,
installs a new first reaction over months.

> Source phrasing is quoted in short form for fidelity. Summarized from the
> book the user provided.

---

## 调用本 skill 时必须做的

1. **Triage 先于开方**——临床案例(创伤/成瘾/精神危机)一律转介;在 Breaking 与
   Building 之间先选对路径(见 §0)。不替用户做"人生要不要变"的决定。
2. **忠于原书,逐字保留**——框架名、术语、原话(见 §1 Frameworks 表里标注
   "author's original naming")**原样保留**,不要用自己的话改写或简化。
3. **台账是内部参考,产出要短**——不要把台账全堆给用户;输出一个能装进口袋的
   句子,然后再解释它为什么这样写。

---

## 0. First move: triage, don't prescribe

Before writing any script, locate the user's problem on the map:

1. **Is it in scope?** Ordinary first-reaction problems = yes. Active
   trauma / PTSD / dissociation, chemical dependency (alcohol, opioids,
   benzodiazepines), active psychiatric crisis, or pure motor-skill
   complexity = **out of scope → refer to professional care.**
2. **Breaking or Building?** Replacing an existing reaction (smoking, the
   snap) = *Breaking*, ~months, one script. Creating a behavior that has
   no moment yet (daily writing) = *Building*, ~a year, a script **stack**.
3. **What's the failure mode?** Match against the Anti-patterns table —
   most stalls are a known, named failure, not a personal defect.

## 1. Frameworks (keep author's original naming — do not paraphrase)

| Name (as written) | One-line | When to use | Exact phrasing |
|---|---|---|---|
| **The half-second** | The pre-deliberation window that decides what you actually *do* | Diagnosing the gap between stated values and behavior | "The story is the cover. The action is the data." |
| **The frequency counter** | The System-1 subsystem that *counts*, never *checks* | Explaining why repetition installs anything, true or not | "System 1 counts, it does not check." |
| **ARISE form** | The five design dimensions: Action / Reason / Identity / Situation / Emotion | The master template for writing any script | "first reactions **arise** in the half-second from these five components" |
| **Identity is the hub** | Edit the identity (hub) and many actions (spokes) shift at once | One old reaction is wired to dozens of situations | "edit the identity… and many actions adjust at once" |
| **Routing** | Whether a sentence reaches the counter or gets vetoed at the door by the conscious mind | Judging if a script *can* install | "Routing is the key term." |
| **Installation vs understanding** | Two kinds of knowing — declarative (books) vs procedural (changes the half-second) | Checking "did I actually learn or just comprehend" | "understanding without installation is not learning" |
| **Breaking vs Building** | Breaking = interception (one cue, one swap); Building = architecture (multi-script, multi-environment) | Scoping the effort of "stop X" vs "start Y" | "Breaking is interception… building is architecture" |
| **The keyboard** | The write-side of the channel others have been writing you on | The closing reframe — you can author your own first reactions | "There is a keyboard on the writer's side of the channel." |
| **Four stages** | Familiarity threshold → Active availability → Installation → Stabilization | Locating how far along the practice is | author's four sub-headings, verbatim |

### 易错点速查（most-mixed-up pairs）

Frameworks 表是"全字典"；这 4 对是 agent 接到任务时**最常读反、合并、搞混**的。
先把这 4 个守住,再去查主表。

| 区分对 | 迷惑之处 | 守住 |
|---|---|---|
| **Breaking vs Building** | 用户说"戒 X"和"建立 X"听起来是一回事 | 破坏 ≈ 1 个 cue + 1 个 swap,数月;建立 = 多脚本 + 多环境,**约 1 年**。量级差 **10×**,别把建立当破坏报时间 |
| **Installation vs understanding** | "我读完了 / 我懂道理" 被误认成"我学会了" | 检验标准:半秒里动作真的换了吗?**没换 = 没学会**。"understanding without installation is not learning" |
| **Universal vs situated** | 总想加 `when [cue]` 显得精确,或反过来总想"我从不…" | 只有行为**真的普世**才用 universal(否则会被 veto 抓到);做不到普世就用 situated 防 veto |
| **Identity recoverability** | 总想加"我是 X" 强调身份 | 动词带身份就别补;补了"我是"反而**打断**脚本的真实感(见 §3 Identity recoverability test) |

*Edge / optional frameworks (mention only if relevant):* **Self tag** (own
voice transmits, external voice doesn't — author flags as provisional);
**Local frequency dominance** (maintenance = win the count at the cue, not
globally).

## 2. Principles (rules that govern every script)

| Name | One-line | When | Exact phrasing |
|---|---|---|---|
| **Action, not mood** | Target a performable action, never a feeling — design rule #1 | Writing every script | "target what the half-second can do, not what the half-second can feel" |
| **Three iron constraints** | Present tense / fits one exhale / in the speaker's own voice | Validating script grammar | "Present tense. Brief. In the speaker's own voice." |
| **Honesty about scope** | Universal vs situated must be honest to your real behavior | Deciding to use "never/always" | "universal scope is for actions whose scope is honestly universal" |
| **Real, not performed emotion** | Felt content must be true and proportionate | Adding emotion to a script | "Performed emotion fails." |
| **The protest is the routing succeeding** | The early "this feels stupid" = the script bypassing the veto = working | Weeks 1–3, when tempted to quit | "The protest is the routing succeeding." |
| **One script at a time** | Start with 1–2; stack monthly | Planning | "one or two scripts at a time for the first six months" |
| **Repeat anywhere** | The counter doesn't weight by location | Killing the "must practice in-situ" myth | "the mechanism is the count, not the cue" |
| **Plan for the upper end** | Expect months-to-a-year (Lally: median 66d, up to 254d) | Setting the timeline | "I'm asking you to plan for the upper end." |
| **Mechanism is value-neutral** | It can install bad identities; safety is on you | Before installing a morally-loaded identity | "the keyboard is unguarded; choose what to install" |
| **Maintenance is continuous** | Install isn't the end; the environment rewrites you daily | After installation | "continuous re-installation… against an environment that is also re-installing" |

## 3. Techniques (the moves, in build order)

| Name | One-line | When | Exact phrasing |
|---|---|---|---|
| **Universal / situated decision** | First decide if the script names a cue context | Step 1 of writing | "the first design decision the speaker makes" |
| **Identity recoverability test** | If the verb carries identity, leave it implicit — don't patch with "I'm…" | Deciding whether to add an identity clause | "Identity must be recoverable… not necessarily written" |
| **Langer-shape because** | A because-clause makes the mind process it as reasoned, not asserted | Script reads as a bare claim the mind wants to fight | "any because-clause increased compliance" |
| **Person + language levers** | Switch to 2nd/3rd person under stress; use L2 to bypass L1-installed reactions | High arousal, or reactions installed in childhood/mother tongue | "if an attacker was installed in one language, test the counter-script in the other" |
| **Four emotion channels** | Invest feeling in: the words / the reason / a remembered specific / the saying | Deepening per-rep encoding | "felt content can live in many places at once" |
| **Environmental design / friction** | Use friction to lower the script's load | Maintenance | "people, places, things" (AA shorthand, quoted) |
| **Coordinated script stacks** | One headline behavior needs several support scripts | Building projects (writing/exercise) | "Six scripts, supporting one headline behavior." |
| **Clearing + Anchoring** | Free the time first, then anchor to a stable cue | Building a behavior that has no moment yet | "build the moment; then, install the script in it" |
| **Felt-sense diagnostic** | "Feels worse" = misrouted; "mildly silly then nothing" = routed right | Real-time test of a draft script | "mildly silly and then nothing, you have routed it right" |
| **Map the voices** | Write down 2–5 inner-attack phrases + their source | Inner-critic work | "Map the voices first." |

## 4. Anti-patterns (name the failure before fixing)

| Name | One-line | Why it fails | Exact phrasing |
|---|---|---|---|
| **Mood target** | "I am calm / wealthy" — targets a feeling | The half-second can't execute a wish | "the half-second cannot execute a wish" |
| **Affirmation routing failure** | "I'm worthy" gets vetoed on the spot | Backfires worst for those who need it | "Power for Some, Peril for Others" |
| **Future tense** | "I will quit" — read as uncertain prediction | The uncertainty becomes a reason to reject | "the future tense routes the script to the conscious mind" |
| **Situation enumeration** | Trying to beat each cue context one by one | Hundreds of cues; it's willpower in disguise | "willpower wearing a different costume" |
| **Performed emotion** | Manufactured peak state | Installs the *rehearsal of trying to feel* | "the count… of the performance" |
| **Reduce frequency too soon** | Big drop right after install | Old reaction returns within a month | "the speaker reduces frequency too soon" |
| **Too many scripts at once** | 15 in month one | Counter bandwidth splits; none installs | "splitting their attention fifteen ways" |
| **Quit at day 14** | Two weeks, no visible change, stop | Median is 66 days; stopped where nothing shows | "stopped at day fourteen of a process whose median length is sixty-six" |
| **Hollow because** | "because abundance is the universe's nature" | Caught by ~the hundredth rep | "fail by the hundredth" |
| **Dishonest universal** | Claims "never" when behavior isn't | Hands the veto a felt-state discrepancy | "the count stays at zero" |
| **Negative script** | "I shouldn't blame" — says only what *not* to do | Under stress the gap fills with the old reaction | "the half-second filled the gap with the old reaction" |
| **Suppressing justified anger** | Using the tool to kill appropriate anger | Suppressed appropriate anger is its own pathology | "Suppressed appropriate anger is its own pathology." |

## 6. The build procedure (use this when the user asks for a script)

1. **Pick the Action** — the spine. A behavior a stranger could verify
   from outside. Small enough to run under stress.
2. **Decide scope** — Universal (no situation) if honestly always; else
   Situated (`when [cue], I [action]`).
3. **Check identity recoverability** — does the verb carry the identity?
   If yes, leave implicit. If no, sharpen the action; add `I'm [identity]`
   only as a last resort.
4. **Add a Reason if needed** — short, honest, survives the 1000th rep:
   `because [reason]`. Not circular with identity.
5. **Invest Emotion honestly** — in words / reason / memory / saying.
   Never performed.
6. **Validate against the three constraints** — present tense, one exhale,
   own voice. Run the felt-sense diagnostic: a good draft feels *mildly
   silly then nothing*, not *worse*.
7. **Set expectations** — repeat anywhere, many times/day; protest →
   irritation is the threshold; plan for months, upper end of the range;
   one or two scripts at a time.

**Worked examples (from the book, as patterns to imitate):**
- `I never smoke.` (action alone, identity implicit, universal)
- `When a bill arrives, I open it within sixty seconds.` (action + situation)
- `When she speaks, I look at her eyes before I answer, because the
  alternative has been costing me my daughter.` (action + situation +
  reason carrying felt stakes)
- `I decide on weeks, not minutes.` (money / volatility, universal)
